# Platera

Platera is a comprehensive, modern Restaurant Management System built with React, TypeScript, Vite, and Tailwind CSS (v4). It's designed to streamline restaurant operations with features ranging from live order tracking and menu management to inventory alerts and advanced sales reporting.

## Features

- **Interactive Dashboard:** Get a bird's-eye view of total revenue, orders, and active tables.
- **Live Orders Kanban:** Real-time drag-and-drop order tracking.
- **Menu Management:** Easily update item details, upload photos, and manage variants/add-ons.
- **Inventory Tracking:** Monitor stock levels and set automated low-stock email alerts.
- **Tables Management:** Manage table occupancy, reservations, and active orders.
- **Reports & Analytics:** Deep dive into sales metrics, category breakdowns, and export data.
- **Staff Management:** Role-based access control and scheduling.
- **Dark/Light Mode:** Seamless, beautiful UI that automatically adapts to your preference.

## Tech Stack

- **Framework:** React 19 + TypeScript + Vite
- **Styling:** Tailwind CSS v4
- **Charts:** Recharts
- **Icons:** Remix Icon
- **Routing:** React Router v6

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-org/platera.git
   cd platera
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
